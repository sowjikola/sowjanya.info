Portfolio

I have brought here my biggest and favorite works as an SAP Security professional.

01

SAP UI Masking Solution

I spearheaded the implementation and deployment of SAP UI Masking solution for a pharmaceutical client, ensuring compliance with GDPR regulations in the EU. My primary responsibilities included gathering business requirements from the client, configuring the UI Masking solution in SPRO, collaborating closely with the ABAP team to customize the solution using BADIs according to client specifications, and constructing roles for UI Masking functionality.

02

GRC Implementation

I executed the deployment of GRC 12 Access Control modules - ARM, ARA, EAM, and BRM, tailored specifically for an energy utility client. My responsibilities included configuring SPRO for GRC 12, developing job-specific GRC roles, crafting test cases, conducting various tests such as Unit, SIT, UAT, and Performance testing. Additionally, I contributed to establishing ownership structures in GRC, generating rulesets, implementing mitigation controls, and other related tasks.

03

SAP Upgrade Project

Engaged in a SAP migration and upgrade initiative for a paper-based manufacturing client, simultaneously upgrading four SAP systems (GRC, ECC, BW, and SCM). Documented all data pertaining to tables AGR*, USOBX_C, USOBT_C pre- and post-execution of SU25 upgrade steps, as per client specifications. Conducted SU25 steps during the upgrade process, aligned SU24 mappings according to SAP standards, and adjusted affected roles alongside other necessary cutover tasks.
